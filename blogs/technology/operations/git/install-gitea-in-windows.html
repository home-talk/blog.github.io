在Windows中安装Gitea

小叮当 2019-08-22 Gitea运维Git

# 下载 gitea

1、先安装 git 2、下载 gitea https://dl.gitea.io/gitea/

# 安装并开启服务

官网教程:https://docs.gitea.io/zh-cn/windows-service/

创建服务

sc create gitea start= auto binPath= "\"C:\gitea\gitea.exe\" web --config \"C:\gitea\custom\conf\app.ini\""
1

启动服务

net start gitea
1

从 Windows 服务中删除

sc delete gitea
1

# 后台运行及开机启动（可忽略）

tips：路径中不能有空格

创建文件 gitea.vbs 填入如下内容，双击运行

set wscriptObj = wscript.createObject("wscript.shell")
wscriptObj.run "C:\Gitea\gitea.exe", 0
1
2

开机自启呢， 创建一个 gitea.vbs 的快捷方式，把它放到自己电脑的启动目录， 我的启动目录在这里（C:\Users\cc\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup），一般将 cc 更换为自己的用户名就能找到了。 （不用快捷方式，直接用 gitea.vbs 也可以）
