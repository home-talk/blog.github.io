使用VS Code开发.NET Core 0002 - 之创建指定版本的项目、类库、控制台程序，并添加应用间的引用

小叮当 2020-11-14 .Net Core后端Dot-net-with-vs-code

# 一、查看当前主机上安装了那些 SDK

.net core 2.1 以及之后新的版本,可以通过如何命令查询

dotnet --list-sdks
1

# 二、使用命令创建项目

1、创建项目文件夹

项目文件夹命名方式使用大驼峰，例如：Time.Man、User、Admin.Api

2、创建 global.json 文件指定 SDK 版本

dotnet new global.json
1

or

dotnet new globaljson --sdk-version 3.0.100
1

3、创建空白解决方案

dotnet new sln
1

4、创建 webapi 项目，并且添加到解决方案中 sln

dotnet new webapi -o Time.Man.Api
1
dotnet sln Time.Man.sln add Time.Man.Api/Time.Man.Api.csproj
1

5、创建 asp.net core 类库

dotnet new classlib -o Common
1

6、创建 asp.net core 控制台程序

dotnet new console -o Test
1
